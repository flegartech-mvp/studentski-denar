# SHORT VIDEO SCRIPT — Študentski Denar

**Format:** 30-second screen recording, no voiceover required (captions work)
**Tone:** Clean, direct, satisfying

---

## Hook — 0 to 5 seconds

**Screen shows:** Browser tab opens cold — no login page, no splash screen, no "sign up with Google." The dashboard loads instantly with an empty state and a single prompt: "Dodaj prvi vnos" (Add your first entry).

**Caption overlay:** "Budget app. No login. No account. No BS."

---

## Show — 5 to 20 seconds

**Screen shows (in sequence, each action takes ~2-3 seconds):**

1. Click "Dodaj prihodek" (Add Income) → type "450" → select category "Štipendija" → hit Save. A green income entry appears in the list.
2. Click "Dodaj odhodek" (Add Expense) → type "12.50" → select "Hrana & Pijača" → hit Save. A red expense entry appears below.
3. Add one more expense: "85" under "Stanarina." Balance at the top updates to €352.50 in real time.
4. Click the "Analitika" tab — a pie chart and a bar chart render instantly showing the category split.
5. Refresh the page. All data is still there.

**Caption overlay during refresh:** "Closes. Reopens. Still there. No server needed."

---

## Close — 20 to 30 seconds

**Screen shows:** The monthly summary view — a clean card showing total income, total expenses, and net balance for the current month. Mouse hovers over the pie chart, which highlights "Hrana & Pijača" with a tooltip.

**Caption overlay:** "Free. Private. Built for Slovenian students."

**Final frame:** App URL or GitHub link centred on screen against a clean background.

---

## Production Notes

- Record at 1280×800 or 1440×900 — wide enough to show both sidebar and main content
- Use a clean browser profile with no extensions visible in the toolbar
- Add sample data before recording the analytics section so charts look populated
- No background music needed — clean UI sounds (click feedback) are enough
